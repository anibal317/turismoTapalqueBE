Create a new city point of interest.

Request Body:

Parameter Type Description
name string Name of the point of interest.
location string Location (coordinates or address).
type number Type identifier for the point of interest.
typeId number Tipo de la atracción, es un enum:

ADMIN = 'ADMIN',

USER = 'USER',

EVENT='EVENTS',

HOSPITALITY='HOSPITALITY',

GASTRONOMY='GASTRONOMY'

Example:

                    POST /city-point-of-interest
                    {
                        "name": "Central Park",
                        "location": "New York",
                        "type": 1
                    }
